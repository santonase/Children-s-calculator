# Дитячий калькулятор 🚀

Ігровий калькулятор для навчання дітей арифметиці: додавання, віднімання, множення, ділення.

## Як грати

Відкрити `index.html` у браузері (або через GitHub Pages). Дитина обирає операцію,
відповідає на приклади з варіантами відповідей, а ракета летить у космос за кожну
правильну відповідь.

## Технології

Звичайний HTML/CSS/JavaScript — без збірки, без залежностей. Можна редагувати
прямо з телефону через GitHub.
