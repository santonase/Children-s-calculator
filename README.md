# Дитячий калькулятор 🚀

Ігровий навчальний застосунок для дітей: додавання, віднімання, множення, ділення, порядок дій і частини числа. Завдання відповідають шкільній програмі (1-4 класи НУШ).

Талісман — **Лис Космік**, лисеня-космонавт, що супроводжує дитину в подорожі космосом.

## Можливості

- **Класи 1-4** з розділами відповідно до шкільної програми
- **Три типи завдань:** таблиця, пропущене число, текстові задачі
- **100 завдань** на тему у вигляді 10 доріжок-станцій, з нескінченною генерацією
- **Пояснення після помилок** — візуальні та текстові підказки
- **Звук, вібрація, озвучення завдань** (Web Audio + Web Speech)
- **Профілі гравців** з аватарами, зірками, рангами
- **Система життів** у стилі Duolingo
- **Магазин скінів ракет**, таблиця лідерів, щоденні стрики
- **Досягнення, щоденні цілі, сундуки-нагороди**
- **Батьківська статистика** (точність по темах, час занять) з математичним замком
- **Дві мови:** українська та англійська (з автовизначенням)
- **Темна і світла теми**
- **Адаптивний дизайн:** телефон, планшет, десктоп
- **PWA:** офлайн-робота, встановлення на головний екран

## Технології

Звичайний HTML/CSS/JavaScript без збірки. Модульна архітектура (js/):
config, i18n (uk/en), storage, logic, generator, sound, hints, badges, stats, monetization, app.

## Плани

- Firebase: синхронізація між пристроями + push-нагадування
- Публікація в Google Play через TWA (Bubblewrap)
- Реальні платежі (Google Play Billing / Stripe)
